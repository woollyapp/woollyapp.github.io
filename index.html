Browse your database
Without having to use any SQL.

See your databases, their tables and even other schemas if you are into that sort of thing. You can grab field values from the table browser but you don't have to worry about accidently overwritting data, the browse view only uses read queries and never issues an update query.

We love Heroku so much
We baked support for it right in.

Heroku is awesome but finding the connection details can be a bit tricky, especially if you are a beginner. So we made it as easy as signing into Heroku, clicking the app you'd like to connect to, and even allowing you to save those details for later. We don't store your Heroku credentials ever, and we keep your connection details in the encrypted system Keychain. Safe and simple, just like Heroku.

Answer your query
Right in the beautiful query interface.

The table browser is read only, but if you want to update your database you can do it from the query browser. It supports syntax highlighting and saving of queries so that you can reuse your queries to save time.
